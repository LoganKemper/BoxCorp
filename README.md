# BoxCorp

**A game about boxes.**

## What is this?

This is a Unity game project for the Wooster Games 'Unity engineer take-home exercise.' I made it in a weekend to demonstrate my Unity and game design skills. And it worked! They hired me. Nice.

Originally created on July 27th, 2025. Then did some project clean-up and added a few features before making the repo public on June 14th, 2026.

## Features
- Box delivery.
- Lots of fun physics objects.
- Particles galore.
- A burnt-out BoxCorp employee.
- A crunchy retro presentation.
- Pleasing beeps **and** boops.
- 3,151 lines of tidy code.
- A boss fight?
